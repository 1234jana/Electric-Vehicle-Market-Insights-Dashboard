# Electric Vehicle Data Analysis

Electric Vehicle (EV) Data Analysis using Excel and Tableau to uncover EV adoption patterns, vehicle distribution, and key trends through data cleaning and interactive dashboards.

## Overview

This project analyzes Electric Vehicle (EV) data to uncover meaningful insights and trends in adoption, vehicle types, and regional distribution. The workflow covers data cleaning and preprocessing in Excel, followed by interactive visualization and dashboard creation in Tableau.

## Tech Stack

* MS Excel — data cleaning, preprocessing, and transformation
* Tableau — data visualization and dashboard creation

## Dataset

The dataset contains information about electric vehicles, including:
* Vehicle Type (BEV, PHEV)
* Model and Manufacturer
* Registration Details
* Location Data
* Range and other specifications

## Project Workflow

1. Data Cleaning (Excel)
   * Removed null and duplicate values.
   * Standardized column names.
   * Filtered relevant data and prepared the dataset for analysis.
2. Data Analysis
   * Identified trends in EV adoption over time.
   * Compared different vehicle types (BEV vs PHEV).
   * Analyzed distribution of vehicles across locations.
3. Dashboard & Visualization (Tableau)
   * Built an interactive dashboard showing EV distribution by type, total vehicle counts, state-wise analysis, and trends over time.

## Key Insights

* Battery Electric Vehicles (BEVs) show higher adoption compared to PHEVs.
* EV usage is concentrated in specific regions.
* A significant growth trend was observed over time.

## How to Run

1. Clone this repository.
2. Open the Excel file to review the data cleaning steps.
3. Open the Tableau file (`.twbx`) to explore the interactive dashboard.

## Files

* Excel file — data cleaning and preprocessing reference.
* Tableau dashboard (`.twbx`) — interactive visualizations of EV adoption trends and insights.
